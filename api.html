AI Messaging & AI Audio Services

AI Messaging: Configure billable AI messaging by specifying a tier and the number of bundles (each bundle = 1000 messages).
- tier: 'standard' provides 2 knowledge bases.
- tier: 'premium' provides 5 knowledge bases.
- bundles is the number of 1000-message increments (e.g., 5 = 5000 messages).

AI Audio Services: Configure by specifying seatLicenses (number of AI Audio Services seat licenses) and additionalHours (number of additional AI transcription hours).

EHR/EMR Integration: Use ehrEmrIntegrationSeats to specify the number of agent users with EHR/EMR integration. 

{
  "aiMessaging": {
    "tier": "premium",
    "bundles": 5
  },
  "aiAudioServices": {
    "seatLicenses": 10,
    "additionalHours": 20
  },
  "ehrEmrIntegrationSeats": 10
}
